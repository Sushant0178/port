Unused Medicine Donation System:

Overview: The Unused Medicine Donation System is an innovative platform designed to collect unused medicines from individuals and distribute them to those in need. Built using Python, Django, SQLite, HTML, CSS, and JavaScript, this system bridges the gap between surplus medicine and underprivileged communities, reducing medical waste and promoting social welfare.

  • User-Friendly Interface: Individuals can easily list and donate unused medicines.
  • Efficient Tracking: Tracks donated medicines and ensures they reach verified beneficiaries.
  • Authentication System: Secure login for donors and administrators to manage donations.
  • Database Integration: Powered by SQLite for efficient data storage and retrieval.
  • Responsive Design: Built with HTML, CSS, and JavaScript for seamless usage on all devices.
